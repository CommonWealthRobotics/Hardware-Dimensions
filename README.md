# Hardware-Dimensions
Hardware dimensions usable in mechanical design.

## Function
Dimensions are stored in  JSON files generated BowlerStudio. 


```JSON
{
"filename" : [
    { 
    "Row1, Col1" : "Row2, Col1",
    "Row1, Col2" : "Row2, Col2",
    },
     { 
    "Row1, Col1" : "Row3, Col1",
    "Row1, Col2" : "Row3, Col2",
    }
    ]
}   
```
